# Walmart Sales Dashboard (Power BI)

## Description  
A Power BI dashboard analyzing Walmart sales data (2010–2012) across 45 stores to uncover trends, store performance, and the impact of economic factors.

## Dataset  
- Source: Kaggle Walmart Sales Dataset  
- Records: ~1,000  
- Features: Weekly Sales, Temperature, Fuel Price, CPI, Unemployment  

## Features  
- KPI metrics (Total Sales, Avg Weekly Sales, Holiday Sales)  
- Monthly sales trends  
- Store-wise performance comparison  
- Holiday vs non-holiday analysis  
- Fuel price vs sales correlation  
- Interactive filters (Year, Store)  

## Tech Stack  
- Power BI  
- Power Query  
- DAX  
- Excel  

## Key Insights  
- Sales peak mid-year and decline towards year-end  
- Majority of sales occur during non-holiday periods  
- Significant variation across store performance  
- Weak negative correlation between fuel price and sales
